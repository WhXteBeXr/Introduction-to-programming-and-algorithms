# Лабораторные работы по C++

Репозиторий содержит выполненные мной учебные лабораторные работы
по дисциплине **«Введение в программирование и алгоритмы»**.

**ВУЗ:** СКФУ (Северо Кавказский Федеральный Университет)

---

## О проекте

В репозитории представлены решения всех лабораторных работ
по **1-му варианту**.

Каждая лабораторная работа оформлена как отдельная программа
и может быть собрана и запущена независимо.

Исходные файлы с условиями заданий находятся в папке `labTasks`.

---

## Сборка и запуск

Проект использует **CMake** и стандарт **C++20**.

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

Запуск нужной лабораторной:
```bash
./lab01
```
